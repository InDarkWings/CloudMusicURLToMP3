# CloudMusicURLToMP3
HTML网易云播放链接转换器

简单易用的网易云外放链接转换器，下载到本地后直接解压后点开index.html到浏览器打开即可，复制分享链接粘贴后点击按钮即可。

在线版：https://wyy-url-to-mp3.pages.dev 或 https://indarkwings.now.cc/

第一个文件夹里是在线版的文件，包括HTML和xml ，而下面的是一开始上传的离线版文件
